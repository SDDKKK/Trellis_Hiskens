# MATLAB Development Guide

## Purpose

Package-scoped MATLAB guidance for Hiskens scientific computing workflows.

## Pre-Development Checklist

- Identify the package you are changing first.
- Read `code-style.md` before editing algorithms or scripts.
- Read `docstring.md` before changing function headers or comments.
- Read `quality-guidelines.md` before claiming completion.
- If the task crosses into Python, also read `../python/index.md`.

## Scope

Use this layer for:

- numerical computing and engineering analysis
- MATLAB scripts that orchestrate domain workflows
- MATLAB functions called from Python or feeding Python outputs
