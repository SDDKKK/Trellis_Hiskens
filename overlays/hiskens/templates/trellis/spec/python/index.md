# Python Development Guide

## Purpose

Package-scoped Python guidance for Hiskens scientific computing workflows.

## Pre-Development Checklist

- Identify the package you are changing first.
- Read `data-processing.md` before touching DataFrame or file I/O logic.
- Read `code-style.md` before editing Python modules or scripts.
- Read `docstring.md` before changing function headers or comments.
- Read `quality-guidelines.md` before claiming completion.
- If the task crosses into MATLAB, also read `../matlab/index.md`.

## Scope

Use this layer for:

- scientific computing scripts
- data preparation and transformation
- reliability and numerical analysis code
- Python glue around MATLAB workflows
