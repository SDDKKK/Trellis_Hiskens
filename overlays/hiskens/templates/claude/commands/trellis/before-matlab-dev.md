Read the MATLAB development guidelines before starting your development task.

Execute these steps:
1. Read `.trellis/spec/matlab/index.md` to understand available guidelines
2. Based on your task, read the relevant guideline files:
   - Code style → `.trellis/spec/matlab/code-style.md`
   - Docstring → `.trellis/spec/matlab/docstring.md`
   - Quality → `.trellis/spec/matlab/quality-guidelines.md`
3. Understand the coding standards and patterns you need to follow
4. Then proceed with your development plan

This step is **mandatory** before writing any MATLAB code.
