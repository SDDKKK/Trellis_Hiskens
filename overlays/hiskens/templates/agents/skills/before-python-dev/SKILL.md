---

Read the Python development guidelines before starting your development task.

Execute these steps:
1. Read `.trellis/spec/python/index.md` to understand available guidelines
2. Based on your task, read the relevant guideline files:
   - Data processing → `.trellis/spec/python/data-processing.md`
   - Docstring → `.trellis/spec/python/docstring.md`
   - Quality → `.trellis/spec/python/quality-guidelines.md`
3. Understand the coding standards and patterns you need to follow
4. Then proceed with your development plan

This step is **mandatory** before writing any Python code.
